# Wiki Springfield

Una página web estática tipo wiki dedicada a los personajes y la familia Simpson. El proyecto ofrece un diseño limpio con navegación intuitiva entre la página principal y las secciones de personajes, con estilos personalizados basados en Bootstrap y CSS.

Wiki Springfield está construido como un sitio informativo y visualmente atractivo para presentar datos clave sobre Homero Simpson y la familia Simpson, ideal para practicar desarrollo web estático con HTML, CSS y Bootstrap.

## Instalación

1. Clona el repositorio en tu máquina local:

   ```bash
   git clone https://github.com/tu-usuario/wiki-springfield.git
   ```

2. Abre la carpeta del proyecto:

   ```bash
   cd wiki-springfield
   ```

3. No se requieren dependencias adicionales: el proyecto funciona como un sitio estático con HTML y CSS.

## Uso

1. Abre `index.html` en tu navegador web.
2. Navega entre las páginas disponibles:
   - `index.html` — Inicio del sitio.
   - `pages/homero.html` — Página de Homero Simpson.
   - `pages/familiasimpson.html` — Página de la familia Simpson.

## Estructura del proyecto

- `index.html` — Página principal del sitio.
- `pages/homero.html` — Detalles del personaje Homero Simpson.
- `pages/familiasimpson.html` — Información sobre la familia Simpson.
- `css/styles.css` — Estilos personalizados del proyecto.
- `css/bootstrap.min.css` — Framework Bootstrap para diseño responsivo.
- `img/` — Imágenes utilizadas en el sitio.
